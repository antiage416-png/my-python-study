# 📊 파이썬 파일 시스템 탐색기 프로젝트

## 🛠 사용 기술
* **Language**: Python 3.10+
* **Libraries**: `pathlib`, `matplotlib`, `json`
* **Tools**: VS Code, Git

## 🚀 주요 기능
1. **파일 탐색**: 특정 폴더 내의 모든 파일 목록을 추출합니다.
2. **용량 계산**: 하위 폴더를 포함한 전체 용량을 Byte 단위로 계산합니다.
3. **시각화**: `matplotlib`을 사용하여 폴더별 용량을 막대그래프로 보여줍니다.
